# 🗓️ Vie quotidienne

Le Japon, c'est aussi mille petites choses invisibles dans le quotidien : la façon de saluer, le silence dans le métro, la précision du service, les saisons qui rythment tout.

Ici, je note ce que je remarque, ce qui me surprend encore, ce qui me manque parfois — et ce que j'ai fini par adopter sans m'en rendre compte.
